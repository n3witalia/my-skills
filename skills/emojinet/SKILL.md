# Emojinet

Communicate exclusively through emoji. Transform any concept, answer, or explanation into visual emoji sequences with absolutely no words or letters.

## When to use this skill

Activate this skill when the user:
- Explicitly requests emojinet or emojinet mode
- Asks to communicate with only emoji
- Says answer using only emoji or similar phrases
- Requests no words, just emoji
- Wants responses in pure emoji format

## Core rules

1. Zero text: Never use letters, words, or text of any kind
2. Only emoji: Communicate exclusively through emoji characters
3. Conceptual mapping: Translate abstract concepts into emoji sequences
4. Maintain meaning: Preserve the core message and intent through visual symbols
5. Creative combinations: Use emoji sequences creatively to express complex ideas

## Communication guidelines

### Simple responses
Map direct concepts to corresponding emoji:
- Agreement: 👍 ✅ 💯
- Disagreement: 👎 ❌ 🚫
- Uncertainty: 🤷 ❓ 🤔

### Complex explanations
Build emoji sequences that tell a story or explain a process:
- Use directional arrows for flow: ➡️ ⬅️ ⬆️ ⬇️
- Use time indicators: 🌅 ☀️ 🌙 ⏰
- Use quantity indicators: 1️⃣ 2️⃣ 3️⃣ or 📈 📉

### Technical concepts
Translate technical ideas into visual metaphors:
- Computer/code: 💻 ⌨️ 🖥️
- Network/internet: 🌐 📡 🔗
- Data/files: 📄 📁 💾
- Security: 🔒 🔐 🛡️

### Emotional expression
Convey tone and feeling through emoji selection:
- Positive: 😊 🎉 ❤️ ⭐
- Negative: 😞 💔 ⚠️
- Excitement: 🚀 🔥 ⚡
- Calm: 🌊 🍃 ☮️

## Workflow

1. Receive user input or question
2. Analyze the core concept or message to convey
3. Select appropriate emoji that represent the concept
4. Arrange emoji in logical sequences
5. Output only emoji—no letters, words, or text whatsoever
6. Continue all subsequent responses in emoji-only format until user explicitly exits the mode

## Edge cases

- Lists or sequences: Use numbered emoji (1️⃣ 2️⃣ 3️⃣) or bullets (• ▪️ ◾)
- Yes/no questions: ✅ or ❌
- Comparisons: Use emoji + ➡️ + emoji or emoji ❌ emoji ✅
- Time-based: 📅 🕐 ⏰ with relevant emoji
- Location-based: 🌍 🗺️ 📍 with context emoji

## Exiting the mode

The skill remains active until the user explicitly requests to return to normal text communication.
